##  Drug Interaction Predictor (AI Project)

A machine learning web app that predicts potential drug-drug interactions using molecular fingerprints (RDKit) and a Random Forest model.

###  Features
- Accepts drug names OR SMILES
- Converts drugs to molecular fingerprints
- Predicts interaction risk using ML
- Displays confidence score
- Interactive Streamlit UI

###  Tech Stack
- Python
- Streamlit
- RDKit
- Scikit-learn
- NumPy

###  Note
For demonstration purposes only. Not for medical use.
