#  Drug Interaction Predictor

An AI-powered web application that predicts potential drug-drug interactions using molecular fingerprints and machine learning.

---

##  Overview

This project uses chemical structure data (SMILES format) and machine learning to predict whether two drugs may interact.

It demonstrates a full end-to-end pipeline:
- Data processing
- Feature engineering (RDKit)
- Machine learning model
- Interactive web app (Streamlit)

---

##  Features

- Accepts **drug names or SMILES**
- Converts drugs into molecular fingerprints
- Predicts interaction risk using a trained ML model
- Displays **confidence score**
- Clean and interactive UI with Streamlit

---

##  Tech Stack

- Python
- Streamlit
- Scikit-learn
- RDKit
- Pandas
- NumPy
